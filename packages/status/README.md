# @frontile/status

Component Library for Ember Octane apps: Status

## Compatibility

- Ember.js v3.16 or above
- Ember CLI v2.13 or above
- Node.js v10 or above

## Installation

```sh
ember install @frontile/status
```

## Documentation

Visit [frontile.dev](https://frontile.dev/) to read the docs
and see live demos.
